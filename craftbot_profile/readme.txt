If have inferior computers:

Copy the craftbot definition files to:
C:\Program Files\Cura 4.11\resources\definitions

Copy the craftbot extruders files to:
C:\Program Files\Cura 4.11\resources\extruders

Copy the craftbot meshes files to:
C:\Program Files\Cura 4.11\resources\meshes

Copy the craftbot images files to:
C:\Program Files\Cura 4.11\resources\images

Copy the craftbot materials files to:
C:\Program Files\Cura 4.11\resources\materials


If have Mac:

Copy to Applications->right click on Cura->ShowPackageContents->Contents->Resources->resources->same files as specified above
